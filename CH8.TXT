SQL> REM REMAINING 3 SET OPERATORS, PERFORMS TWO COMMON TASKS : 1 REMOVES DUPLICATE ROWS 2 SORTS ROWS IN ASCENDING ORDER
SQL> 
SQL> REM  UNION : IT WILL ONLY PERFORM ABOVE TWO COMMON TASKS.
SQL> 
SQL> REM INTERSECT : 1 TWO COMMON TASKS 2 REMOVE ALL NON-COMMON ROWS FROM RESULT SET
SQL> 
SQL> REM MINUS : 1 TWO COMMON TASKS 2 REMOVES ALL COMMON ROWS FROM FIRST RESULT SET AND WILL PROVIDE REMAINING ROWS OF FIRST
SQL>
